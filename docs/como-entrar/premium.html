Minecraft Premium

¡Bienvenido al tutorial para entrar a SaS: 1.5!

Existen 2 formas de entrar al servidor desde Minecraft:

Para PC de bajos recursos

Configuración de SaS​

Puedes entrar al servidor desde el launcher de Minecraft, para ello debes seguir los siguientes pasos:

  1. Abre el launcher de Minecraft e inicia sesión con tu cuenta de Microsoft.

  2. Una vez dentro del launcher, presiona Instalaciones.

  3. Presiona en Nueva Instalación

  4. Selecciona la versión 1.19.4 y presiona Crear

  5. Selecciona la versión recién creada y presiona en Jugar

  6. Una vez dentro del juego, presiona Multijugador

  7. Presiona Agregar Servidor

  8. En Dirección IP escribe mc.sascraft.club y en Recursos del server presiona hasta que diga Activado y presiona Aceptar

  9. Finalmente entra al servidor presionando Entrar al servidor. Abre el chat con la tecla T y presiona que tienes cuenta premium. Luego haz click derecho en algún cartel para entrar al servidor deseado. Puedes volver a selecciona un servidor utilizando el comando /servers

Para PC con más de 8 GB de RAM

Configuración de SaS:​

  1. Descarga aquí el siguiente pack premium y descomprímelo en tu escritorio.
  2. Encontrarás un archivo llamado 'MultiMC', el cual contiene el launcher que utilizarás. También tiene el archivo 'OpenJDK21U', que es la mejor versión de Java para jugar. Recomiendo que primero instales el archivo OpenJDK21U y luego abras el archivo MultiMC.
  3. El archivo que utlizarás como Launcher es 'MultiMC.exe'. Puedes crear un acceso directo en tu escritorio para acceder más rápido.
  4. Una vez abierto el launcher aparecerá el icono de SaS. Haz doble click y te pedirá que agregues una cuenta de Microsoft.
  5. Cuando presiones el botón te aparecerá una ventana con un link y un código. Copia el código y pégalo en la ventana que abrirá el link. Luego inicia sesión con tu cuenta de Microsoft.
  6. Luego de que hayas iniciado sesión, vuelve a hacer doble click en el icono de SaS para que se abra el juego. Ahora con el juego abierto deberás presionar Descargar Pack y añadir los packs al juego.
  7. Finalmente, presiona en Entra a SaS: 1.5. Abre el chat con la tecla T y presiona que tienes cuenta premium. Luego haz click derecho en algún cartel para entrar al servidor deseado.

Puedes volver a selecciona un servidor utilizando el comando /servers
